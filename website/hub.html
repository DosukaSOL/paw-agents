Six Integrated Views

Everything in One Window

Each view is purpose-built for a specific workflow. Switch instantly between monitoring, commanding, extending, and automating your agents.

📊

Dashboard

Real-time stats, agent chat, and at-a-glance metrics for your entire fleet.

  • Action count, uptime, plugin stats
  • Live chat with your agent via WebSocket
  • Cross-app sync status indicator
  • Quick-switch between AI providers
🎯

Mission Control

Monitor all agents, track tasks, view system metrics, and manage alerts in real-time.

  • Agent registry with online/idle status
  • Task queue with priority management
  • CPU, memory, request metrics
  • Alert system with severity levels
💻

CLI

A full terminal interface inside Hub. Run commands, inspect state, and manage everything.

  • Agent status, mode switching
  • Plugin management commands
  • Sync and workflow controls
  • Custom command registration
🧩

Plugins

Browse, install, and manage plugins that hook into the agent pipeline.

  • DeFi Alerts — price & whale tracking
  • Analytics — usage & cost metrics
  • GitHub — issues, PRs, monitoring
  • Scheduler — cron jobs & recurring tasks
⚡

Workflows

Build and execute multi-step automation workflows from built-in or custom templates.

  • Price Alert flow (3 steps)

  • Onboarding sequence (4 steps)

  • AI provider selection (7 providers)

  • OAuth2 & tenant configuration
  • Theme and display preferences

Cross-App Memory Sync

Every action, every memory, every conversation — synced across all 11 channels in real-time. Start on Telegram, continue on Hub, finish on Discord.

🧠

Shared Memory

Key-value store with TTL and versioning. Set a memory on any channel, read it from any other. Automatic conflict resolution.

📋

Action Tracking

Every action records its source channel. View in Hub Settings which actions came from Telegram, Discord, API, or any other channel.

🔗

Session Continuity

Cross-app sessions link your conversations across channels. Shared context means the agent always knows your full history.

Get Started

Install PAW Hub

Hub runs as an Electron app alongside the PAW Gateway. With Ollama, no API keys are needed — it's completely free. 

# Clone PAW Agents
$ git clone https://github.com/DosukaSOL/paw-agents.git
$ cd paw-agents

# Install + build core
$ npm install && npm run build

# (Optional) Install Ollama for free local AI
$ curl -fsSL https://ollama.com/install.sh | sh
$ ollama pull gemma4

# Launch Hub
$ cd desktop && npm install && npm start

Requires Node.js 20+ and the PAW Gateway running. Ollama is optional but recommended for free, private AI.
